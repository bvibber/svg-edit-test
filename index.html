A round-trip test suite for SVG-edit, to test SVG images from Wikimedia Commons

Please see the SVG-edit round-trip test suite discussion as part of the larger goal of getting SVG-edit used on Wikipedia.

Use with Firefox and be patient for the page to fully load all the test images. In the image output, the first image is the original SVG loaded from Wikimedia Commons, the second is the file as loaded into and then saved from SVG-Edit running in an iframe, and the third is a diff produced by drawing the SVGs onto canvas elements, then subtracting one from the other in raw pixel manipulation. This shows up areas that differ very visibly. A fully black box and a difference score of 0 means success.
